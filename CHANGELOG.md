## [v0.1.0] - 2026-07-19

### Highlights

- Initial release: batteries-included distribution composing omnidevx-core, omni-github, and omni-openai behind one import path

### Added

- Re-exported canonical types, event/mode/severity constants, and provider constructors (Claude Code, Codex, git, GitHub) under a single `omnidevx` import path ([`20cd110`](https://github.com/plexusone/omnidevx/commit/20cd110))
- `NewDefault` for composing local Claude Code and Codex collectors against the current machine's default stores ([`20cd110`](https://github.com/plexusone/omnidevx/commit/20cd110))
- `Engine` runs an arbitrary set of `core.Collector`s over one request; one collector failing does not stop the others, and errors are joined with their source named ([`66ad85e`](https://github.com/plexusone/omnidevx/commit/66ad85e))
- `Events` flattens collection results from multiple collectors into a single event slice ([`66ad85e`](https://github.com/plexusone/omnidevx/commit/66ad85e))

### Dependencies

- `github.com/plexusone/omnidevx-core` v0.1.0 for the canonical event model ([`af19e93`](https://github.com/plexusone/omnidevx/commit/af19e93))
- `github.com/plexusone/omni-github` v0.5.0 for the GitHub DevX collector ([`af19e93`](https://github.com/plexusone/omnidevx/commit/af19e93))
- `github.com/plexusone/omni-openai` v0.6.0 for the Codex CLI collector ([`af19e93`](https://github.com/plexusone/omnidevx/commit/af19e93))

### Documentation

- README documenting the batteries-included distribution, the OmniDevX/OmniDXI distinction, available collectors, and usage examples ([`57753a8`](https://github.com/plexusone/omnidevx/commit/57753a8))
- README shields for CI, lint, SAST, godoc, mkdocs, visualization, and license ([`233ef90`](https://github.com/plexusone/omnidevx/commit/233ef90))
- Fix README badges that pointed at omnidevx-core instead of omnidevx, and update the Collectors table to include Git and GitHub ([`1625a5e`](https://github.com/plexusone/omnidevx/commit/1625a5e))
- MkDocs site (Material theme, matching omni-github/omni-openai/omnidevx-core) with a home page, Collectors reference, and v0.1.0 release notes ([`b1bdcbc`](https://github.com/plexusone/omnidevx/commit/b1bdcbc))

### Tests

- Multi-collector aggregation, per-collector failure isolation with error joining, Add/Collectors chaining, context cancellation, and provider-constructor composition tests ([`fb8fb75`](https://github.com/plexusone/omnidevx/commit/fb8fb75))

### Infrastructure

- GitHub Actions workflows (go-ci, go-lint, go-sast-codeql), Dependabot config, and golangci-lint settings shared with the other omni-* repos ([`bca2222`](https://github.com/plexusone/omnidevx/commit/bca2222))
